Features of Gift Wrapping in Magento 2 

Gift Wrapping for Individual Items: 

Magento 2 allows customers to choose gift wrapping for individual items in their cart, rather than for the entire order. This feature can be configured in the “Stores” section under the “Checkout” tab in the Magento 2 admin panel.

Gift Wrapping Message: 

Magento 2 also provides the option to include a gift wrapping message along with the wrapped items. This feature can be enabled in the “Stores” section under the “Checkout” tab in the Magento 2 admin panel.

Gift Wrapping Templates: 

Magento 2 allows you to create custom gift wrapping templates to use for your gift wrap products. These templates can be designed to match your store’s branding and can be uploaded in the “Stores” section under the “Design” tab in the Magento 2 admin panel.

Gift Wrapping Report: 

Magento 2 provides a gift wrapping report in the admin panel that shows the total number of gift wrap orders and the amount of revenue generated from gift wrap sales. This report can be accessed in the “Reports” section under the “Sales” tab in the Magento 2 admin panel.

Gift Wrapping for Virtual Products: 

It’s important to note that gift wrapping is not available for virtual products, such as digital products or services, as they cannot be physically wrapped.”

Steps to add and configure gift wrap in Magento 2 

Step 1: Open Command Line Interface (CLI)

To get started, open your preferred command line interface (CLI) or terminal window. Make sure you have SSH access to your Magento 2 installation, as you will need to run commands to configure gift wrap.

Step 2: Enter Magento 2 Root Directory

Navigate to the root directory of your Magento 2 installation using the ‘cd’ command. For example:

bash

cd /var/www/html/magento2

Replace ‘/var/www/html/magento2’ with the path to your Magento 2 installation.

Step 3: Run the Gift Wrap Command

Magento 2 provides a built-in command to add gift wrap functionality. Run the following command in your CLI:

arduino

php bin/magento setup:upgrade

This command will update your Magento 2 installation and enable the gift wrap feature.

Step 4: Configure Gift Wrap Options

Once the gift wrap feature is enabled, you can configure the gift wrap options. Run the following command:

bash

php bin/magento giftwrap:configure

This command will prompt you to enter the gift wrap options, such as the gift wrap title, price, and tax class. You can customize these options according to your business needs.

Step 5: Set Gift Wrap as a Configurable Product Option

Next, you need to set gift wrap as a configurable product option, so customers can select it during the checkout process. Run the following command:

bashÂ 

php bin/magento giftwrap:assign

This command will prompt you to select the configurable product to which you want to assign the gift wrap option. Choose the desired product and save the configuration.

Step 6: Customize Gift Wrap Design

You can also customize the design of your gift wrap to match your store’s brand. Magento 2 allows you to specify different wrapping papers, ribbons, and notes. Run the following command:

bash

php bin/magento giftwrap:design

This command will prompt you to enter the design options, such as the wrapping paper color, ribbon style, and note template. You can customize these options to create a unique and personalized gift wrap design for your customers.
