How to Add Dynamic Links to the Footer in Magento 2

Published on: February 21, 2023

Magento 2 is a highly popular e-commerce platform that provides a plethora of features and functionalities to setup of Magento 2 store. One of the crucial features that contribute to improving website navigation is the ability to add links to the footer section. By incorporating dynamic links to the footer, businesses can enhance the user experience and make their site more accessible.

Adding links to the footer is a critical aspect of website design and user experience, as it allows for swift and effortless navigation to other essential pages on the website. These footer links can be static or dynamic, making it easy for merchants to modify and update them as required. By offering clear and accessible footer links, businesses can enhance the user experience and increase the accessibility of their website to customers.

Aside from footer links, Magento 2 offers an array of customization options, including the ability to select from various themes, create custom pages and layouts, and integrate with a wide range of third-party extensions and plugins. These advanced features make Magento 2 a potent and adaptable e-commerce platform that can enable businesses to succeed online.

This blog will cover how to add dynamic links to the footer in Magento 2. The process is simple, straightforward, and can be accomplished in just a few steps.

Steps to Add Dynamic Links to the Footer in Magento 2 

Step 1: Create a New Module

The first step is to create a new module that will contain the code for adding dynamic links to the footer. To create a new module, you will need to create a new directory in the app/code directory with the following structure:

app/code/[Vendor]/[Module]

Replace [Vendor] with your company name or your name, and [Module] with the name of your module. Inside the module directory, create a registration.php file and add the following code:

<?php

\Magento\Framework\Component\ComponentRegistrar::register(

    \Magento\Framework\Component\ComponentRegistrar::MODULE,

    '[Vendor]_[Module]',

    __DIR__

);

Next, create a module.xml file in the same directory and add the following code:

<?xml version="1.0"?>

<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:Module/etc/module.xsd">

    <module name="[Vendor]_[Module]" setup_version="1.0.0"/>

</config>

Step 2: Create a Block Class

The next step is to create a block class that will contain the code for adding the dynamic links to the footer. Create a new directory in the module directory with the following structure:

app/code/[Vendor]/[Module]/Block

Inside the Block directory, create a FooterLinks.php file and add the following code:

<?php

namespace [Vendor]\[Module]\Block;

use Magento\Framework\View\Element\Html\Link\Current;

class FooterLinks extends Current

{

    protected $_template = '[Vendor]_[Module]::footer_links.phtml';

    public function getLinks()

    {

        $links = [];

        $links[] = [

            'label' => __('Link 1'),

            'url' => $this->getUrl('link-1'),

            'title' => __('Link 1'),

            'liClass' => 'link-1'

        ];

        $links[] = [

            'label' => __('Link 2'),

            'url' => $this->getUrl('link-2'),

            'title' => __('Link 2'),

            'liClass' => 'link-2'

        ];

        return $links;

    }

}

Step 3: Create a Template File

The next step is to create a template file that will render the dynamic links in the footer. Create a new directory in the module directory with the following structure:

app/code/[Vendor]/[Module]/view/frontend/templates

Inside the templates directory, create a footer_links.phtml file and add the following code:

<?php

/** @var $block \[Vendor]\[Module]\Block\FooterLinks */

$links = $block->getLinks();

?>

<ul>

    <?php foreach ($links as $link): ?>

        <li class="<?= $link['liClass'] ?>">

            <a href="<?= $link['url'] ?>" title="<?= $link['title'] ?>">

                <?= $link['label'] ?>

            </a>

        </li>

    <?php endforeach; ?>

</ul>

Step 4: Add the Block to the Footer

The final step is to add the block to the footer of your Magento 2 store. To do this, create a new layout file in the module directory with the following structure:

app/code/[Vendor]/[Module]/view/frontend/layout

Inside the layout directory, create a default.xml file and add the following code:

<?xml version="1.0"?>

<page xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:View/Layout/etc/page_configuration.xsd">

    <body>

        <referenceContainer name="footer">

            <block class="[Vendor]\[Module]\Block\FooterLinks" name="[Vendor]_[Module]_footer_links" template="[Vendor]_[Module]::footer_links.phtml"/>

        </referenceContainer>

    </body>

</page>

This code adds a reference to the footer container and adds a new block to it with the class and template we created earlier.

Step 5: Clear the Cache

After completing the above steps, clear the cache by running the following command from the Magento 2 root directory:

php bin/magento cache:clean
