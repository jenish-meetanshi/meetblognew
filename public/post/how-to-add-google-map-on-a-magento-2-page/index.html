How to Add Google Map in Magento 2 Page? A Step-by-Step Guide

Why should you embed Google Map to your website?

The goal of integrating Google Maps onto a website is to make it easy for customers to find a physical store. Although online shopping has become increasingly popular, many shoppers still prefer in-store shopping.

Adding a Google Map to a website offers more than just informing customers about the store location. It also:

  • Simplifies the process of searching for store locations by allowing customers to use the map directly on the site
  • Provides full business contact information, such as phone number, website, reviews, and ratings
  • Increases the time visitors spend on the site, providing opportunities to convert them into customers
  • Recommend and highlight nearby points of interest, parks, and restaurants, which can help visitors better understand the store’s physical location
  • Functions as a fully functional Google Map app on smartphones, ensuring customers can always find the business

Increases shopper’s trust and confidence in the business by displaying the exact location and contact information on Google Maps

Builds brand visibility and credibility by highlighting multiple physical store locations on the map.

In summary, integrating Google Maps into an eCommerce website offers numerous benefits, including making it easier for customers to find the physical store, providing contact information, and increasing shopper trust and confidence.

How to add Google Map in Magento 2 store?

Typically, Magento 2 store owners often choose to include Google Maps on their About Us or Contact Us pages to display their physical store locations and provide directions to online visitors.

Add Google Map in Magento 2 with default Page Builder

In Magento 2 Commerce Edition, merchants have the option to add a Google Map block to their About Us and Contact Us pages using the Page Builder features. However, before adding the map, they must create a billing account or project on Google Maps Platform and activate the Google Maps Javascript API. Once these steps are completed, they will receive an API key, which they can then use to configure the map in Magento 2. Additionally, to optimize page load speed, it’s recommended to combine Javascript files, and this article will guide you on How to Combine Javascript files in Magento 2.

Step 1: Add the Google Map API Key in the Store Backend

  • Log in to the Admin dashboard and navigate to Stores > Settings > Configuration > Content Management > Advanced Content Tools.
  • Paste your API key in the Google Maps API Key field and click the Test Key button to verify its correctness and usability.
  • If you are unsure how to obtain the API key, follow the instructions from Google Maps.

Step 2: Add Google Map to a Magento 2 Page

  • Go to Content > Pages in the left sidebar.
  • Select the page where you want to add the Google Map,
  •  click Edit in the Action column. The Page Builder will appear in the Content section.
  • In the Page Builder panel, select Media. Drag and drop the Map option to the stage. If Google Maps Platform is not configured for your store, the map will not be displayed.

Step 3:To add store locations to the map:

  • Move the mouse to the map container to access the toolbox.
  • Click on the Setting icon to open the Edit Map page.
  • In the upper-right corner, click the Add Location button.
  • Fill in the location name and the exact coordinates in Google Maps.
  • Add any additional relevant information to associate with the map location.

To find the coordinates of a location:

  • Open the Google Maps website.
  • Search for the location.
  • The coordinates can be found in the URL.

Step 4: Customize the Map’s Appearance

  • Access the configuration settings by navigating to Stores > Settings > Configuration > General > Content Management > Advanced Content Tools.
  • Click on the “Create Map Style” button under the Google Maps Style field.
  • Choose “Create a Style” and set the map’s appearance to your desired style (e.g., feature density, theme, etc.).
  • When finished, click on the “Finish” button.
  • Export the completed style as either a JSON code or URL, then add it to the Magento 2 configuration.
  • Paste the code or URL into the Google Maps Style field.
  • Save the configuration changes by clicking on the “Save Config” button to display the customized map on your website.
