How to Add Multiple Products to Cart in Magento 2 

Published on: February 14, 2023

Magento 2 is a well-known e-commerce platform that offers a wide range of tools and functionalities for managing and creating online stores. Its flexible, scalable, and customizable nature makes it a perfect platform for businesses of any size.

One of the key features of Magento 2 is its shopping cart, which is integral to the customer purchasing process. The shopping cart allows customers to add multiple products to their cart, including the ability to create bundle products in Magento 2, and purchase them in a single transaction, which can enhance the overall shopping experience and save time.

This blog will delve into the process of bulk adding products to the cart in Magento 2, a primary feature of Quick Order M2. Moreover, we will discuss how this feature can benefit online store owners in boosting sales and enhancing customer satisfaction. Additionally, we will touch upon various options available for customizing the shopping cart in Magento 2, such as custom product options and automatic discounts and promotions.

Whether you are a seasoned Magento 2 user or a newcomer, this guide will provide you with a comprehensive understanding of how to add multiple products to the cart in Magento 2 and how to leverage this feature for your online store.

Benefits to add Multiple Products to the Cart in Magento 2

Here is a detailed explanation of the benefits of adding multiple products to the cart in Magento 2:

Saves Time and Effort: 

Adding multiple products to the cart in a single transaction saves the customer time and effort by avoiding the need to make multiple transactions for each product.

Improves the Checkout Process: 

With all the products in the cart, the checkout process becomes more streamlined, as the customer only needs to enter their shipping and payment information once. This can make the checkout process faster and more convenient for the customer.

Offers a Better Overview of the Purchase: 

By having all the products in the cart, customers can have a better overview of their purchase, including the total cost, the products included, and the quantities of each product. This can help customers make informed purchasing decisions.

Enhances the Customer Experience: 

The process of adding multiple products to the cart can play an important role in the overall customer experience. A fast and convenient process for adding multiple products to the cart can help create a positive shopping experience for customers.

Increases Average Order Value: 

By making it easier for customers to purchase multiple products in a single transaction, it can lead to an increase in average order value. This can benefit the online store by generating more revenue per transaction.

Reduces Cart Abandonment: 

Cart abandonment is a common issue in e-commerce, but having a streamlined and convenient process for adding multiple products to the cart can reduce the likelihood of customers abandoning their carts. This can lead to increased sales and customer loyalty for the online store.

Prerequisites:

Before we start adding multiple products to the cart programmatically, there are a few prerequisites that need to be met. These include:

1. A working Magento 2 installation with access to the codebase

2. A basic understanding of PHP programming

3. Access to a text editor or IDE

Steps to add multiple products to cart in Magento 2

Step 1: Creating a PHP file

To add multiple products to the cart in Magento 2 programmatically, we need to create a PHP file in the codebase. Open your text editor or IDE and create a new PHP file in the root directory of your Magento 2 installation. You can name the file “add-multiple-products-to-cart.php” or any other name you prefer.

Step 2: Setting up the Magento 2 Object Manager

Next, we need to set up the Magento 2 Object Manager. The Object Manager is responsible for creating and managing objects in Magento 2. We will use it to load the products we want to add to the cart. Add the following code to your PHP file:

php

use Magento\Framework\App\Bootstrap;

require __DIR__ . '/app/bootstrap.php';

$bootstrap = Bootstrap::create(BP, $_SERVER);

$objectManager = $bootstrap->getObjectManager();

Step 3: Loading the products

Now that we have set up the Object Manager, we can load the products we want to add to the cart. We can load products by their SKUs or IDs. In this example, we will load products by their SKUs. Add the following code to your PHP file:

php

$productSkus = ['SKU1', 'SKU2', 'SKU3']; // Replace with your product SKUs

$products = [];

foreach ($productSkus as $productSku) {

    $product = $objectManager->create('Magento\Catalog\Model\Product');

    $productId = $product->getIdBySku($productSku);

    $product->load($productId);

    $products[] = $product;

}

Step 4: Adding the products to the cart

We have loaded the products we want to add to the cart, and now we can add them to the cart. We will use the Magento 2 Cart object to add products to the cart. Add the following code to your PHP file:

php

$cart = $objectManager->create('Magento\Checkout\Model\Cart');

foreach ($products as $product) {

    $params = array(

        'product' => $product->getId(),

        'qty' => 1

    );

    $cart->addProduct($product, $params);

}

$cart->save();

Step 5: Running the PHP script

Save the PHP file and upload it to your Magento 2 installation. You can run the PHP script by navigating to the URL of the PHP file in your web browser. For example, if you named the file “add-multiple-products-to-cart.php” and uploaded it to the root directory of your Magento 2 installation, you can access it at:

bash

http://your-magento-2-installation.com/add-multiple-products-to-cart.php
