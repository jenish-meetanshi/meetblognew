How to Configure Layered Navigation in Magento 2 Store: A Complete Guide

The central objective of Magento 2’s Layered Navigation is to empower customers to swiftly find their desired product by filtering options based on attributes such as price, color, material, and others. This enhances the shopping experience by enabling customers to locate products that meet their specific needs.

Configure Price Filter in Layered Navigation

In Magento 2, you can customize your layered navigation settings, including price navigation, by navigating to Stores > Configuration > Catalog > Catalog > Layered Navigation in the general settings.

You will find two fields that must be set in the layered navigation configuration:

1. Display Product Count – determines if the number of products matching product attributes, price, or category will be displayed next to the layered navigation options.

2. Price Navigation Step Calculation – defines the method used to calculate the price navigation step.

Magento 2 offers three methods for calculating the price navigation step:

1. Automatic (equalize price ranges) – determines the price navigation steps automatically based on the price range of products in a group.

2. Automatic (equalize product count) – calculates the price navigation steps automatically based on the number of products in a group. This option includes additional fields, such as:

  •  Display Price Interval as One Price – to be enabled when all prices in a specific interval are equal.
  • Interval Division Limit – the number of products that will not be divided into subintervals.

3. Manual – allows customers to set prices for layered navigation manually. In this case, you can configure the following options:

  •  Default Price Navigation Step – determines the number of products included in each step.
  •  Maximum Number of Price Intervals – defines the maximum number of price intervals available in Magento 2 layered navigation.

Configure Layered Navigation with Filterable Attributes

Magento 2’s Layered Navigation does not include product attributes by default. To allow customers to filter products by attributes, you must manually configure it.

It is important to note that not all product attributes can be added to layered navigation. Only attributes with dropdown, multiple select, or price input types are eligible for inclusion.

If there is no suitable attribute to add to layered navigation, you can create a new attribute and assign it to the attribute set.

To add product attributes to Magento 2’s Layered Navigation, follow these steps:

1. Go to Stores > Attributes > Products and select the attribute you want to add to layered navigation.

Note: You will notice that the ‘Use in Layered Navigation’ section has already been added to the product attribute options.

2. Navigate to the Storefront Properties section of the Attribute Information.

3. In the ‘Use in Layered Navigation’ dropdown, select either ‘Filterable (with results)’ to display the number of products in the category associated with this attribute, or ‘Filterable (no results)’ if you don’t want to display the product count.

4. To display this attribute on the search results pages, select ‘Yes’ in the ‘Use in Search Results Layered Navigation’ option.

5. Remember to click ‘Save Attribute’ once you’ve made the necessary changes.

How to display Layered Navigation on Magento 2 Category Page?

To display the layered navigation menu on a Magento 2 category page:

1. Go to Calatog > Categories.

2. Choose the desired category and enable the ‘Anchor’ option in its Display Settings.

The Layered Navigation menu in Magento 2 offers a convenient way for customers to search for products by filtering available options. This enhances the shopping experience and improves engagement in your store. You can display the layered navigation on any category page by enabling the Anchor option in the Display Settings. Additionally, you can adjust the placement of the menu by configuring the category design in the Design section.
