How to Configure robots.txt File in Magento 2

Published on: February 03, 2023

Magento 2 is a robust e-commerce platform that offers a variety of features and functionalities to setup Magento 2 and managing an it. One of the essential elements of an online store is the robots.txt file, which plays a crucial role in directing search engines on how to crawl and index the site’s pages. The robots.txt file is a simple text file that resides in the root directory of a website and informs search engine bots which pages or sections of the site should not be crawled or indexed.In this blog post, we’ll explain how to configure the robots.txt file in Magento 2.

Steps to Configure robots.txt File in Magento 2

Step 1: Access the Magento 2 Admin Panel

To start, log in to your Magento 2 admin panel.

Step 2: Navigate to the Store Configuration

In the Magento 2 Admin Panel, click on the “Stores” tab and then click on “Configuration.”

Step 3: Go to the Search Engine Robots Section

Under the “General” tab, click on the “Design” option. Then, expand the “Search Engine Robots” section.

Step 4: Edit the Robots.txt File

In the “Search Engine Robots” section, you can edit the robots.txt file directly in the text editor. You can add or remove rules as needed.

Step 5: Save the Changes

Once you have made the necessary changes, click on the “Save Config” button to save your changes.

Step 6: Verify the Changes

To verify the changes, you can use a tool like the robots.txt checker to see if the changes have been implemented correctly.

Conclusion

In conclusion, having a properly configured robots.txt file is critical for the success of your online store. It helps you control how search engines crawl your website, which in turn affects your website’s visibility and search engine ranking. With the steps outlined in this blog, you can easily configure the robots.txt file in Magento 2 and ensure that search engines are indexing your website correctly. Remember to always verify the changes made to the robots.txt file and to have a clear understanding of its purpose before making any modifications.
