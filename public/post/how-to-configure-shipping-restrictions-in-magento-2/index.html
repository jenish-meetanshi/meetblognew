How to Configure Shipping Restrictions in Magento 2

Published on: February 14, 2023

Shipping restrictions are an essential aspect of managing an e-commerce store efficiently. They enable you to control the delivery of your products and ensure that only eligible customers receive them. By setting criteria such as location and weight, you can streamline the shipping process, minimize errors, and reduce returns.

Magento 2 offers a user-friendly interface for configuring shipping restrictions, including the option to auto select the shipping method in Magento 2 for customers during checkout. With a wide range of shipping restriction options available, such as geographical location and shipping carrier, you can create customized restrictions that meet your store’s and customers’ unique needs.

This comprehensive guide provides step-by-step instructions on how to configure shipping restrictions in Magento 2. Whether you’re a new user or a seasoned expert, this guide equips you with the information and tools you need to configure shipping restrictions for your store quickly and efficiently.

Steps to configure shipping restrictions in Magento 2

Step 1: 

Connect to your Magento 2 installation via SSH.

Step 2: 

Run the following command to list all of the available shipping methods:

python

bin/magento setup:di:compile

Step 3: 

Identify the shipping methods you want to restrict, and note their corresponding carrier codes. For example, if you want to restrict the Flat Rate shipping method, the carrier code would be “flatrate”.

Step 4: 

Run the following command to create a new shipping method restriction:

bash

bin/magento config:set carriers/[CARRIER_CODE]/active 0

Replace “[CARRIER_CODE]” with the carrier code of the shipping method you want to restrict.

Step 5: 

Run the following command to create a new shipping method restriction that only allows shipping to specific countries:

bash

bin/magento config:set carriers/[CARRIER_CODE]/specificcountry [COUNTRY_CODE_1,COUNTRY_CODE_2,...]

Replace “[CARRIER_CODE]” with the carrier code of the shipping method you want to restrict, and “[COUNTRY_CODE_1,COUNTRY_CODE_2,…]” with a comma-separated list of the country codes you want to allow shipping to.

Step 6: 

Run the following command to create a new shipping method restriction based on the order subtotal:

bash

bin/magento config:set carriers/[CARRIER_CODE]/min_order_subtotal [MIN_ORDER_SUBTOTAL]

Replace “[CARRIER_CODE]” with the carrier code of the shipping method you want to restrict, and “[MIN_ORDER_SUBTOTAL]” with the minimum order subtotal required to qualify for this shipping method.

Step 7: 

Repeat steps 4-6 for each shipping method you want to restrict.

Step 8: 

Run the following command to apply the new shipping method restrictions:

bash

bin/magento cache:clean

And that’s it! You have successfully configured shipping restrictions in Magento 2 using commands. With these restrictions in place, you can now control the delivery of your products and ensure that they are only shipped to eligible customers.
