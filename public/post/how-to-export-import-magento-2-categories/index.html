How to Export/Import Magento 2 Categories

Published on: June 07, 2023

If you are managing a large Magento 2 store with many categories, you know that creating and managing them manually can be time-consuming and prone to errors. Thankfully, Magento 2 provides a convenient way to export and import categories using CSV files, which can save you time and effort in managing your store.

In this article, we’ll provide you with a step-by-step guide on how to export and import Magento 2 categories using CSV files.

What Does a CSV File Consist of ?

A CSV file, short for Comma Separated Values, is a type of plain text file that contains data separated by commas. The file consists of rows and columns, with each row representing a record and each column representing a field in that record. For instance, a CSV file might include a list of products with columns for the product name, description, price, and SKU. Each product would be listed as a separate row in the file, with the product name in the first column, description in the second column, price in the third column, and SKU in the fourth column. The values in each column are separated by a comma, making it easy to import the data into a database or spreadsheet program for further analysis.

Exporting Magento 2 Categories:

Step 1: Log in to your Magento 2 store admin panel and navigate to the “System” menu in the main menu bar.

Step 2: Click on “Data Transfer” and select “Export” from the drop-down menu.

Step 3: In the Export Settings section, select “Stock Sources” as the Entity Type.

Step 4: Configure the export settings as per your requirements. You can choose the file name and the location where you want to save the exported file.

Step 5: In the “Export File Format” section, select “CSV” as the file format. Also set Field Enclosure as Checked.

Step 6: When you are exporting products, it is possible to add filters by the list of attributes. Now if you set word check in the SKU field only products having word Check in their SKU will be exported.  Then click on the “Continue” button to proceed.

Step 7: In the next section, you can select the categories you want to export. You can filter the categories by store view, status, and visibility.

Step 8: Once you have selected the categories, click on the “Continue” button to start the export process.

Step 9: Wait for the export process to complete. You can download the exported CSV file from the location you specified in Step 4.

Importing Magento 2 Categories:

Step 1: Log in to your Magento 2 store admin panel and navigate to the “System” menu in the main menu bar.

Step 2: Click on “Data Transfer” and select “Import” from the drop-down menu.

Step 3: In the “Import Settings” section, select “Stock Sources” as the Entity Type.

Step 4: Click on the “Choose File” button and select the CSV file that contains the categories you want to import.

Step 5: Configure the import settings as per your requirements. You can choose the behavior for existing categories, whether to replace or append the data.

Step 6: Click on the “Check Data” button to validate the CSV file.

Step 7: If there are no errors in the CSV file, click on the “Import” button to start the import process.

Step 8: Wait for the import process to complete. Once the process is complete, you can see the imported categories in your Magento 2 store.

Conclusion:

Exporting and importing Magento 2 categories through CSV files can significantly reduce the time and effort required to manage your store. Simply follow the outlined steps to export and import your categories with ease. By optimizing your category management, you can enhance the user experience of your store and boost your sales.
