How to Generate XML and HTML Sitemaps in Magento 2

Published on: February 14, 2023

Introduction:

Sitemaps are a crucial component of any website as they serve as a guide for search engines to effectively crawl and index all pages of your site.

A well-organized sitemap provides valuable information about the structure and content of your website, simplifying the process for search engines to comprehend and rank your pages. XML sitemaps, specifically, aid search engines in rapidly discovering and indexing all of your pages, while HTML sitemaps present a user-friendly overview of your website’s structure, simplifying navigation for visitors.

In Magento 2, generating XML and HTML sitemaps is a simple task that can be accomplished with ease via the backend of the platform, after you set up Magento 2. This article will guide you through the step-by-step process of How to Generate XML and HTML sitemaps in Magento 2, as well as highlight the advantages of doing so.

Generating XML Sitemap in Magento 2:

Step 1: Log in to the Magento 2 backend

The first step is to log in to your Magento 2 backend by accessing the URL that is provided to you when you installed Magento 2. Enter your username and password to log in to the Magento 2 backend.

Step 2: Go to the Sitemap section

Once you are logged in, navigate to the “Marketing” section from the left-side menu and then select “Sitemaps”.

Step 3: Click on Add Sitemap

Click on the “Add Sitemap” button to create a new sitemap.

Step 4: Enter Sitemap Information

In the “Add Sitemap” section, enter the following information:

Filename: Enter a filename for your sitemap. The filename must end with .xml.

Path: The path is automatically generated based on the filename you have entered.

Store View: Select the store view for which you want to generate the sitemap.

Priority: Set the priority of the pages in the sitemap. The priority of the pages is a value between 0.0 and 1.0.

Frequency: Select the frequency with which you want the sitemap to be updated.

Step 5: Generate the XML Sitemap

Once you have entered all the required information, click on the “Save & Generate” button to generate the XML sitemap. The sitemap will be generated and stored in the specified path.

Now from the Admin Panel, please go to Stores > Configuration > Catalog > XML Sitemap

Now fill up the information by clicking on the generation setting tab you will now set up and configure the schedule of your xml sitemap over here.

Creating XML and HTML sitemaps in Magento 2 can significantly enhance your website’s visibility to search engines.
