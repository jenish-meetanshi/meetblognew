How to Install a Theme in Magento 2

Published on: January 16, 2023

The visual aesthetics of a web-based business store plays a vital role in its overall appeal and engagement with customers. It is imperative to have an attractive website when selling anything online to retain and engage customers on the site.

Fortunately, there is an abundance of free Magento 2 themes to choose from, and with this tutorial, you can easily learn how to install the theme of your choice.

Although the process of installing a theme in Magento 2 is relatively simple, there are still several steps that you must follow to ensure a successful installation in Magento 2. To make things easier for you, we have prepared a step-by-step guide to walk you through the installation process.

Steps to Install a Theme in Magento 2

STEP: 1 Install it Manually

Firstly download the free theme you want then unzip the theme files and here you will find 2 directories app and pub.

Now go to the root directory of Magento 2 and upload the app and pub folders over there.
Now run the following command,

Go to the root directory of your Magento 2 store and also connect your store via SSH

By running the following command you can upgrade your setup,

php bin/magento setup:upgrade

By running the following command you can deploy your static content,

php bin/magento setup:static-content:deploy


STEP: 2

Next, log in to the Magento 2 admin panel and navigate to the “Content” menu. From there, select “Design” and then “Configuration.”


STEP: 3
Click on the “Edit” button under the Action section.


STEP: 4

Now In the Default Theme, you can see Applied Theme Option, and a dropdown button available over there click on it and you will get the options of themes. Select one of them.


STEP: 5
Select the website or store view where you want to use the new theme and then select the theme you just installed from the “Applied Theme” drop-down menu.

STEP: 6
Finally, click on the “Save” button to apply the changes and activate your new theme.

Conclusion

Congratulations on successfully learning how to install a theme in Magento 2. You should now have your new theme installed and activated on your Magento 2 store.

We highly recommend testing a new theme on a development environment before applying it to a live site. This will enable you to identify and resolve any potential issues that may arise before your visitors encounter them.

Moreover, it is always advisable to back up your site before making any significant changes to your website. This will ensure that you can quickly restore your website to its previous state in the event of any unforeseen issues during the theme installation process.
