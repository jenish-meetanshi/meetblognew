How to Install Extension in Magento 2 Manually

Published on: January 30, 2023

Installing extensions in Magento 2 manually is a simple process that can be completed in just a few steps. Whether you’re looking to add new functionality to your online store or improve the overall user experience, installing extensions can be a great way to enhance your website. In this guide, we’ll walk you through the process of manually installing extensions in Magento 2, so you can start taking advantage of the many features and tools available to you.

Magento Marketplace by Adobe Commerce offers multiple extensions from different providers to help grow your business. you can also install magento extensions if needed.

In this blog post, we will walk through the process of how to install extension in Magento 2 manually. 

Steps to Install Extension in Magento 2 Manually 

Step 1: Download the Extension

The first step for how to install extension in Magento 2 manually is to download the extension package. You can usually find extension packages on the Magento Marketplace or the developer’s website. Make sure to download the package that is compatible with your version of Magento 2.

Step 2: Unzip the Extension Package

Once you have downloaded the extension package, unzip it to a directory on your computer. This will reveal the extension’s files and folders.

Step 3: Upload the Extension to Your Magento 2 Installation

The next step is to upload the extension’s files and folders to your Magento 2 installation. To do this, connect to your Magento 2 installation using an FTP client such as FileZilla. Navigate to the “app/code” directory and create a new directory with the name of the extension’s company. Inside this directory, create another directory with the name of the extension. Now upload the extension’s files and folders to this directory.

Step 4: Enable the Extension

Now that the extension’s files are uploaded to your Magento 2 installation, you need to enable the extension. To do this, log into the Magento 2 admin panel and navigate to “Stores > Configuration > Advanced > Advanced”. Under the “Disable Modules Output” section, find the extension you just installed and click on the “Enable” button.

Step 5: Run the Setup Upgrade

The final step is to run the setup upgrade. This step is required to update the database and make the extension work properly. To do this, navigate to the root directory of your Magento 2 installation and run the following command:

php bin/magento setup:upgrade

After running this command, the extension should be successfully installed and enabled on your Magento 2 installation.

Note: In case of how to install extension in Magento 2 manually you are facing any issues with the above steps, you should check for the extension’s documentation for specific installation instructions or reach out to the developer for support.

Conclusion

In conclusion, how to install extension in Magento 2 manually is a straightforward process that can be easily completed by following the steps outlined in this guide. By taking the time to install extensions, you’ll be able to take full advantage of the many features and tools available to you and can improve the overall user experience of your online store. Keep in mind that installing extensions may require you to make changes to your website’s code, so it’s important to be familiar with Magento 2’s file structure and know how to properly modify the files.
