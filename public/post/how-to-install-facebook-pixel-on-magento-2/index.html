The first step is to create a Facebook Pixel ID, which is a unique identifier that helps Facebook track and analyze user behavior. You can do this by heading over to your Facebook Ads Manager and clicking on “Pixels” in the “Events Manager” section. From there, simply click “Create a Pixel” and follow the prompts to set it up.

Once you’ve created your Pixel ID, it’s time to add the Pixel code to your website. This can be done by navigating to the “Content” section in your store’s backend, clicking on “Configuration,” and selecting the store view to which you want to add the Pixel code. Then, scroll down to the “HTML Head” section and paste the Pixel code into the “Scripts and Style Sheets” box. Remember to save your changes, and you’re done!

How to set up Facebook Pixel in Magento 2?

Step 1: To create a Facebook Pixel:

1. Open the Facebook Ads Manager.

2. Go to the Pixel tab and click on “Create a Pixel.”

3. Enter a name for your pixel and your website URL (optional). Keep in mind that each account is only allowed to create one pixel, but you can add more by upgrading to Business Manager.

4. Set up events and confirm.

5. Retrieve your Pixel ID for future use.

6. Connect your Facebook Pixel with your Google Tag Manager account.

7. Install the pixel code by selecting “Add code using a partner integration” and “Magento” as the option.

8. Place the pixel code in the header of each page on your website. For more detailed instructions, refer to the pixel implementation guide.

Note: If you already have an older version of the conversion tracking pixel, you need to create a new one as Facebook has combined the conversion tracking pixel and the custom audience pixel into one Facebook Pixel.

Step 2: To install Magento 2 Google Tag Manager and integrate it with a Facebook Pixel:

1. Use the Magento 2 Google Tag Manager extension as a tool to quickly add a Facebook Pixel to your website. This extension easily integrates with Magento 2.

2. Follow these steps to download and install the extension:

  •  Log in to your Magento admin account and go to Stores > Configuration > WebGuru extensions > Google Tag Manager > Settings.
  •  Choose “Yes” to enable the module.

3. Retrieve the Container ID from your Google Tag Manager account and enter it in the “Google Tag ID” field in the General Configuration section of the Magento 2 Google Tag Manager extension.

4. Expand the Facebook Pixel section in the Magento 2 configuration and enable the feature.

  •  Choose the ID variable based on product ID or product SKU.
  •  Enter the Pixel ID you received after creating your pixel into the “Facebook Pixel ID” field. If this field is left blank or filled with an incorrect ID, your Facebook Pixel will not be able to track and report data on your Facebook ads and website.

5. Save your configuration. The Facebook Pixel has now been successfully integrated with your website.

Note: If you already host your website on Magento 2, you can use Magento 2 Google Tag Manager as a viable tool to add a Facebook pixel quickly as the Facebook Pixel seamlessly integrates into the module.
