How to Install Magento 2.4 in Windows 10

Published on: January 16, 2023

The utilization of an open-source e-commerce platform is prevalent across the business landscape, as it provides a diverse range of features and functions suitable for companies of any size. 

However, installing version 2.4 of this platform on a Windows 10 operating system can present several challenges. Nevertheless, through proper guidance and instructions, the installation process can be carried out smoothly. 

This tutorial aims to deliver a step-by-step guide on how to install Magento 2.4 in Windows 10, thereby facilitating the setup of your online store and enabling you to begin selling your products swiftly.

Comprehensive Guide to Magento 2.4 Installation on Windows 10

Below are the step-by-step instructions to guide you on how to install Magento 2.4 on Windows 10.

Before we proceed, you should have the following requirements:

  1. Windows 10 OS
  2. PHP 7.4 or later installed
  3. MySQL or MariaDB installed
  4. Composer installed

Now, let’s get started on installing Magento 2.4 in Windows 10.

Step 1: Download and Install Composer

Composer is a dependency manager for PHP that is used to install Magento. To install Composer, follow these steps:

1. Go to https://getcomposer.org/ in your web browser.

2. Click the “Download” button.

3. Download the “Composer-Setup.exe” file and run it.

4. Follow the instructions on the installer to complete the installation.

Step 2: Download Magento 2.4

You can download the latest version of Magento 2.4 from the official Magento website. Follow these steps to download Magento:

1. Go to https://magento.com/tech-resources/download in your web browser.

2. Select “Magento Open Source” and choose the latest version.

3. Click the “Download” button and save the file to your computer.

Step 3: Install Magento 2.4

To install Magento 2.4, follow these steps:

1. Extract the downloaded Magento file to your web server root directory.

2. Open the Command Prompt and navigate to the Magento directory using the following code:

bash
cd C:xampphtdocsmagento

3. Use the following code to install Magento:

composer install

4. Next, use the following code to install Magento:

php
php bin/magento setup:install --base-url=http://localhost/magento --db-host=localhost --db-name=<db_name> --db-user=<db_user> --db-password=<db_password> --admin-firstname=<admin_firstname> --admin-lastname=<admin_lastname> --admin-email=<admin_email> --admin-user=<admin_username> --admin-password=<admin_password> --language=en_US --currency=USD --timezone=America/Chicago --use-rewrites=1

Replace the values in <> with your own values.

Step 4: Configure Magento 2.4

After installation, you can configure Magento by accessing the admin panel at http://localhost/magento/admin. Log in using the admin credentials you specified during the installation.
