How to Install Magento 2.4 Using Composer

Published on: January 19, 2023

Installing Magento 2.4 using Composer is a straightforward process that can be completed in a few simple steps. A composer is a tool for dependency management within Magento 2, that allows you to download the libraries of Magento 2. Developers follow the long manual process to download Magento 2 and they also face difficulty in managing it. This is where Composer comes in for a stress-free experience. In this blog post, we will walk you through the process of how to install Magento 2.4 using composer, so you can have your new e-commerce platform up and running in no time.

Steps to Install Magento 2.4 Using Composer


STEP: 1

First, you will need to have Composer installed on your system. If you do not have Composer installed, you can download it from the official website.

STEP: 2

Once Composer is installed, open up a terminal or command prompt and navigate to the directory where you would like to install Magento 2.4.

STEP: 3

Next, run the following command to install the Magento 2.4 software:

composer create-project --repository-url=https://repo.magento.com/ magento/project-community-edition=2.4.0

This command will download and install the latest version of Magento 2.4 (2.4.0) in the current directory.

STEP: 4

After the installation is complete, navigate to the Magento 2.4 installation directory and run the following command to set up the file permissions:

find var generated vendor pub/static pub/media app/etc -type f -exec chmod g+w {} + && find var generated vendor pub/static pub/media app/etc -type d -exec chmod g+ws {} + && chown -R :<web server group> . && chmod u+x bin/magento

STEP: 5

Next, open your web browser and navigate to your Magento 2.4 installation. You will be prompted to enter your database connection information, as well as your admin user details. Once you have entered this information, click the “Next” button.

STEP: 6

The final step is to run the Magento 2.4 installation script. This can be done by running the following command:

bin/magento setup:install

This command will install and configure the Magento 2.4 software on your system.

Make sure to check the official documentation for the most accurate and updated instructions.
