How to Move Cart Total Below Cart Items in Checkout Page in Magento 2

In this article, we will guide you step-by-step on how to move the Cart Total section below the Cart Items section on the Checkout Page in Magento 2. By default, the Cart Total is displayed above the Cart Items on the Checkout Page in Magento 2. However, if you want to change the position of the Cart Total section and display it below the Cart Items section, follow the instructions provided in this guide. So, let’s get started on how to move cart total below cart items in checkout page in Magento 2.

How to Move Cart Total Below Cart Items in Checkout Page in Magento 2: A Step-by-Step Guide

Are you looking to change the default configuration of the Magento 2 checkout page and move the cart total below the cart items? This can easily be achieved by following a few simple steps. In this guide, we will provide you with a comprehensive tutorial on how to move the cart total below the cart items in the checkout page in Magento 2. By the end of this tutorial, you will have the knowledge and skills to make this change to your Magento 2 checkout page. So, let’s get started!

Step 1: Create an Override File

In order to move the cart total, you need to override the checkout_index_index.xml file in your module or theme, depending on your specific requirements. If you want to create the file for your theme, you need to create it in the following path:

app/design/frontend/Vendor/theme/Magento_Checkout/layout/checkout_index_index.xml

If you want to create the file for your module, you need to create it in the following path:

app/code/Vendor/Module/view/frontend/layout/checkout_index_index.xml

Step 2: Edit the XML File

After you have created the override file, you need to edit the XML code. You can add the following code to the file to move the cart total below the cart items:

<page xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" layout="1column" xsi:noNamespaceSchemaLocation="urn:Magento:framework:View/Layout/etc/page_configuration.xsd">

    <body>

        <move element="checkout.cart.totals" destination="checkout.cart.items" after="-" />

    </body>

</page>

Step 3: Clear the Cache

After you have made the necessary changes, you need to clear the cache. You can clear the cache by going to the Magento 2 admin panel and navigating to System > Cache Management. From there, you can select the Flush Magento Cache button to clear the cache.

Step 4: Test the Changes

Finally, you can test the changes by visiting the checkout page and checking to see if the cart total is now displayed below the cart items.
