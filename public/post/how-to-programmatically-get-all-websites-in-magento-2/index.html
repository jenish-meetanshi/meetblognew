How to Programmatically Get All Websites in Magento 2

Published on: February 03, 2023

Magento 2 is a robust and flexible e-commerce platform that provides a wide range of features and tools for developers to create and manage online stores. As a developer, you may need to retrieve a list of all websites in Magento 2 for various purposes, such as assigning categories, updating website settings, or creating new stores. In this blog, we will dive into the details of how to programmatically get all websites in Magento 2. We will be using the Magento\Store\Model\StoreManagerInterface class, which is a built-in class in Magento 2 that provides access to all store-related data and functionality. This class enables us to retrieve a list of all websites available in the Magento installation and perform various operations on them. By the end of this blog, you will have a clear understanding of how to programmatically get all websites in Magento 2. Additionally, you may also want to know how to install an extension manually in Magento 2, which is a useful skill for developers to have.

Prerequisites

1. Magento 2 installed and running

2. Basic knowledge of Magento 2 development

Steps to programmatically get all websites in Magento 2

Step 1: Create a module

To start, we need to create a module in Magento 2. You can create a module using the following steps:

– Create a folder with the name of your module in app/code.

– Create etc, Controller, Model and view folders in your module folder.

– Create a registration.php file in your module folder with the following code:

php

\Magento\Framework\Component\ComponentRegistrar::register(

    \Magento\Framework\Component\ComponentRegistrar::MODULE,

    'Vendor_ModuleName',

    __DIR__

);

- Create a module.xml file in etc folder with the following code:

lua

<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:Module/etc/module.xsd">

    <module name="Vendor_ModuleName" setup_version="1.0.0">

    </module>

</config>

Replace Vendor and ModuleName with your vendor and module name.

Step 2: Get a list of all websites

To get a list of all websites in Magento 2, we need to use the Magento\Store\Model\StoreManagerInterface class.

Add the following code to your controller file:

php

<?php

namespace Vendor\ModuleName\Controller;

use Magento\Framework\App\Action\Action;

use Magento\Framework\App\Action\Context;

use Magento\Store\Model\StoreManagerInterface;

class Index extends Action

{

    /**

     * @var StoreManagerInterface

     */

    private $storeManager;

    /**

     * Index constructor.

     * @param Context $context

     * @param StoreManagerInterface $storeManager

     */

    public function __construct(

        Context $context,

        StoreManagerInterface $storeManager

    ) {

        $this->storeManager = $storeManager;

        parent::__construct($context);

    }

    /**

     * Get all websites

     *

     * @return void

     */

    public function execute()

    {

        $websites = $this->storeManager->getWebsites();

        foreach ($websites as $website) {

            echo $website->getName().' (ID: '.$website->getId().')<br>';

        }

    }

}

Step 3: Test

Now, run the following URL in your browser to test your code:

perl

http://your-magento-base-url/module-name/index/index

Replace your-magento-base-url and module-name with your Magento base URL and module name.
