How to Reindex in Magento 2 from Command Line

Published on: January 31, 2023

Reindexing in Magento 2 is the process of updating the data in your store’s index to ensure that it is accurate and up-to-date. If you make changes to your store’s data, such as adding new products or updating existing ones, you will need to reindex to reflect those changes in your store’s search results and catalog pages. There are 2 ways to reindex in magento 2 from the admin panel and other by command line. In this article, we will provide a detailed guide on how to reindex in Magento 2 from command line.

Steps to Reindex in Magento 2 from Command Line

Step 1: Connect to your Magento 2 server

The first step to reindexing in Magento 2 from the command line is to connect to your Magento 2 server. You can use a terminal or a command prompt to access your server.

Step 2: Navigate to the Magento 2 root directory

Once you are connected to your Magento 2 server, navigate to the root directory of your Magento 2 store. This is the directory that contains the “app”, “bin”, and “pub” directories.

Step 3: Run the Reindex Command

From the Magento 2 root directory, run the following command to reindex your store:

python
php bin/magento indexer:reindex

Step 4: Wait for the Reindexing Process to Complete

Wait for the reindexing process to complete. This may take a few minutes, depending on the size of your store’s data.

Step 5: Verify the Reindexing Process

After the reindexing process is complete, you can verify that the reindexing process was successful by running the following command:

python
php bin/magento indexer:info

This command will display the status of all the indexes in your store. The status of the reindexed indexes should be “Ready”.

Conclusion

Reindexing in Magento 2 is an important process that ensures the accuracy of your store’s data. By following the steps outlined in this article, you can easily reindex your Magento 2 store from the command line. If you experience any issues with the reindexing process, contact Magento 2 support for assistance. It is always a good idea to backup your store’s database and files before making any changes to your store, such as reindexing.
