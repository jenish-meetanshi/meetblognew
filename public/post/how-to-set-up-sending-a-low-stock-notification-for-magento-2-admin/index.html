How to Set Up Sending a Low Stock Notification for Magento 2 Admin: A Step-by-Step Guide

Manage low-quantity notifications in the default configuration

A warning message may be received by a manager when the available stock of a product drops to a critical level. Magento 2 admin allows the trader to set up these alerts in several areas of the admin panel:

1. Notify for Quantity Below field in the Stores configuration menu under Settings > Configuration > Catalog > Inventory > Product Stock Options sets a default threshold for all items on the site.

2. Advanced Inventory Notify for Quantity Below field (found under Catalog > Products > specific product > Advanced Inventory) overrides the website/store level setting and applies to all sources of the product.

3. Notify Quantity fields (located in Catalog > Products > specific product > Assigned Sources section) are the highest priority setting. The merchant can set a unique threshold for each source of a specific product.

Magento’s Stock Management: Deduction of Global or Overriding Quantity

Magento subtracts either the global or the overriding quantity from the total available stock to determine the salable quantity.

The REST low-quantity notification endpoints allow merchants to manage their values set in the Notify Quantity fields.
