How to Unlock Admin User in Magento 2

Published on: February 02, 2023

If the administrator of an e-commerce store using Magento 2 accidentally loses access to their admin user account or gets locked out of the backend, it can be challenging to regain control. Magento 2 offers several features and customization options for online stores, but sometimes, unlocking the admin user is necessary. This blog will guide you on how to unlock admin user in Magento 2 by accessing the database, locating the user record, updating the user status, clearing the cache, and logging back into the backend.

Steps to Unlock Admin User in Magento 2

Step 1: Access the Database

The first step in unlocking an admin user in Magento 2 is to access the database. You can do this using the following command in your terminal or command prompt:

css
mysql -u [database_username] -p[database_password] [database_name]

Replace [database_username], [database_password], and [database_name] with the credentials for your Magento 2 database.

Step 2: Find the User Record

Once you have accessed the database, you will need to find the record for the admin user account that you want to unlock. You can do this using the following SQL command:

sql
SELECT * FROM `admin_user` WHERE `username`='[admin_username]';

Replace [admin_username] with the username for the admin user account that you want to unlock.

Step 3: Update the User Status

Once you have located the user record, you will need to update the user status to unlock the account. You can do this using the following SQL command:

go
UPDATE `admin_user` SET `is_active` = '1' WHERE `username`='[admin_username]';

Replace [admin_username] with the username for the admin user account that you want to unlock.

Step 4: Clear the Cache

After updating the user status, it is important to clear the Magento 2 cache. You can do this using the following command:

bash
php bin/magento cache:clean

Step 5: Log in to the Backend

With the admin user account unlocked and the cache cleared, you should now be able to log in to the Magento 2 backend using the unlocked admin user account.

Conclusion

In conclusion, unlocking an admin user in Magento 2 is a straightforward process that can be completed in a few simple steps. By accessing the Magento 2 database, finding the user record, updating the user status, clearing the cache, and logging in to the backend, you can regain access to your admin user account.By having the knowledge of how to unlock an admin user, you can quickly regain access to your admin account in case of any unexpected lockouts or access issues. It is important to keep the database and user records updated to avoid such situations in the future.
