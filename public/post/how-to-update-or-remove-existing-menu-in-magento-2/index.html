How to Update or Remove Existing Menu in Magento 2

Published on: February 15, 2023

Magento 2 is an open-source e-commerce platform that offers an array of features and functionalities for merchants to manage their online store. To setup Magento 2 is a straightforward process with its user-friendly interface, making it easy to customize and configure the store to meet specific business needs. One crucial aspect of an online store is the navigation menu, which acts as a roadmap for customers to find the products they want. A well-designed navigation menu can significantly enhance the user experience and increase the chances of a customer making a purchase.

In this blog, we will go over the steps involved in how to update or remove existing menu in Magento 2. Whether you want to change the name of a menu item, add new categories, or delete outdated menus, this guide will help you make the necessary changes in no time. With the flexibility and scalability of Magento 2, you can ensure your online store stays ahead of the curve and meets the evolving needs of your customers.

Prerequisites:

To follow this tutorial, you should have a basic knowledge of Magento 2 and its file structure. You should also have a basic understanding of PHP.

Updating an Existing Menu in Magento 2:

Magento 2 provides a user-friendly interface to create and update menus. To update an existing menu in Magento 2, follow the below steps:

Step 1: Log in to the Magento 2 Admin panel.

Step 2: Navigate to the Content > Elements > Menus.

Step 3: Select the menu that you want to update.

Step 4: Make the required changes to the menu.

Step 5: Click on the Save Menu button to save the changes.

Alternatively, you can also update the menu programmatically by updating the XML file of the menu. To update the menu programmatically, follow the below steps:

Step 1: Create a custom module or use an existing module.

Step 2: Create a new XML file with the below code.

xml

<?xml version="1.0"?>

<menu xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:module:Magento_Backend:etc/menu.xsd">

    <update id="Magento_Catalog::catalog" title="Catalog" module="Magento_Catalog" sortOrder="10" resource="Magento_Catalog::catalog"/>

</menu>

In the above XML code, the id attribute specifies the ID of the menu item that needs to be updated, and the title attribute specifies the new title of the menu item.

Step 3: Create a new file menu.xml in the etc/adminhtml directory of your module and add the below code.

xml

<?xml version="1.0"?>

<config xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance" xsi:noNamespaceSchemaLocation="urn:magento:framework:Module/etc/module.xsd">

    <module name="Vendor_Module" setup_version="1.0.0">

        <sequence>

            <module name="Magento_Backend"/>

        </sequence>

    </module>

</config>

In the above code, replace Vendor_Module with the name of your module.

Removing an Existing Menu in Magento 2:

To remove an existing menu in Magento 2, follow the below steps:

Step 1: Log in to the Magento 2 Admin panel.

Step 2: Navigate to the Content > Elements > Menus.

Step 3: Select the menu that you want to remove.

Step 4: Click on the Delete button to remove the menu.

Alternatively, you can also remove the menu programmatically by removing the XML file of the menu. To remove the menu programmatically, follow the below steps:

Step 1: Identify the XML file of the menu that you want to remove. The XML file can be found in the following directory:

xml

app/code/[Vendor]/[Module]/view/adminhtml/menu.xml

Step 2: Delete the XML file of the menu.

Best Practices for Updating or Removing Menus in Magento 2

Keep the Navigation Simple: It is important to keep the navigation menu simple and easy to understand, so that customers can easily find the products they are looking for.

Test Your Changes: Before making changes to your menus, it is a good idea to test the changes on a staging site to make sure that the changes have the desired effect.

Keep Your Menus Up-to-Date: Regularly review and update your menus to reflect changes in your product catalog, and to remove unused menus.
