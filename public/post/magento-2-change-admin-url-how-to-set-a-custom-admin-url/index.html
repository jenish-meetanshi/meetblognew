Magento 2 Change Admin URL: How to Set a Custom Admin URL

Published on: February 14, 2023

Magento 2 is a popular e-commerce platform that stands out for its broad range of customization options. One critical aspect of ensuring the security of any online store is to secure the backend or admin panel from unauthorized access. Magento 2 provides a vital feature that enables users to customize the default admin URL with a preferred URL of their choice. This additional security layer makes it significantly harder for hackers to guess and access the admin panel, thereby protecting the website from potential cyber attacks. 

This blog post takes an in-depth look at the process of Magento 2 Change Admin URL: How to Set a Custom Admin URL, highlighting the necessary steps in detail to provide a comprehensive understanding of the procedure. 

After reading this post, you will have the knowledge and confidence to secure your online store’s admin panel with a custom URL, enhancing the overall security of your e-commerce website.

Why Change the Magento 2 Admin URL?

Hackers are aware of the default Magento 2 admin URL, making it easier for them to guess and gain unauthorized access to the backend. However, by changing the default URL to a custom one, you can considerably increase the difficulty for hackers to guess and access the backend. This simple yet effective security measure can significantly enhance the security of your website, providing an added layer of protection against potential cyber attacks.

Benefits of Changing Magento 2 Admin URL

Changing the default admin URL offers several benefits that can enhance the security and overall user experience of your website. 

Firstly, it significantly increases the security of your website by making it harder for hackers to guess and access the backend, thereby reducing the risk of cyber attacks.

Secondly, opting for a custom URL allows you to select a relevant and memorable URL that aligns with your website and brand, making it easier for users to remember and access your admin panel.

Lastly, customizing your backend URLs enables you to organize your backend sections in a way that makes sense for your business, facilitating easier and more efficient access to relevant sections of the backend. By implementing these simple but effective measures, you can improve the overall functionality, security, and organization of your website.

Tips for Choosing a Custom URL

Make it memorable: Opt for a memorable URL that relates to your business.

Avoid common terms: Refrain from using common terms like “admin,” “login,” and “backend” since they are popular and simple to guess.

Use a mix of letters and numbers: Employ a combination of letters and numbers to create a more complex URL that is harder to guess.

Keep it short: Shorten the URL to make it easier to remember and less prone to typing errors.

Method 1: Create a custom Admin URL from the Magento Admin

Step 1: Login to the Magento 2 backend

To change the Magento 2 admin URL, first, log in to the backend of your Magento 2 store. You can access the backend by adding “/admin” to your Magento 2 website URL.

Step 2: Go to the Configuration section

Once logged in, navigate to the “Stores” menu and click on “Configuration.”

Step 3: Change the Admin URL

Under the “Advanced” section, click on “Admin” and then select “Security.” You will see the “Admin Base URL” section, where you can change the default URL to a custom URL.

Step 4: Save the Changes

After making the changes, don’t forget to save the configuration. To save the changes, click on the “Save Config” button at the top right corner of the page.

Step 5: Clear Cache

Finally, clear the cache to reflect the changes. Go to the “System” menu and select “Cache Management.” Click on the “Flush Magento Cache” button and you’re done.

Conclusion

Great job! You have successfully followed the steps to change the admin URL in Magento 2 and added an extra layer of security to your online store. Having a custom URL makes it harder for hackers to guess and access the backend, which is crucial for the safety of your website.

However, in case you forget your admin URL or get locked out for any reason, there’s no need to worry. You can easily unlock admin user in Magento 2 by following the guidelines provided in the Magento 2 user guide. Alternatively, you can also contact your hosting provider for assistance.

Method 2: Change the Admin Path from Command Line

In addition to changing the Magento 2 admin URL through the backend, you can also change it through the command line. Here are the steps to change the Magento 2 admin URL from the command line:

Step 1: Connect to your server

Use an SSH client such as PuTTY to connect to your server.

Step 2: Navigate to the Magento 2 root directory

Change to the root directory of your Magento 2 installation.

Step 3: Run the following command

python

php bin/magento setup:config:set --backend-frontname=new_admin_url

Replace "new_admin_url" with your desired custom URL.

Step 4: Clear cache:

Clear the Magento 2 cache using the following command:

python

php bin/magento cache:clean

5: Verify the changes

Verify the changes by logging into the backend using the new URL.

Conclusion 

Congratulations! You have successfully changed the Magento 2 admin URL using the command line. This efficient process allows you to modify the URL without accessing the backend, saving you time and effort.

Please note that changing the admin URL requires caution, and it is essential to have a backup of your website before making any adjustments.
